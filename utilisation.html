Guide d'utilisation

Maîtrisez toutes les fonctionnalités de votre WH-1000XM5.

1

Allumage & appairage

Maintenez le bouton power 2s, puis sélectionnez le casque dans vos paramètres Bluetooth.

2

Réduction de bruit

Appuyez sur le bouton NC/AMB pour basculer entre ANC, mode bruit du vent ou mode transparent.

3

Commandes tactiles

Glissez vers l'avant/arrière pour changer de piste, touchez deux fois pour play/pause, couvrez l'oreillette pour parler.

4

Application Sony

Téléchargez Sony Headphones Connect pour personnaliser l'égaliseur et les fonctions.

Speak-to-Chat : le casque met automatiquement la musique en pause quand vous parlez.
Autonomie record : 30h avec ANC ; 3min de charge = 3h d'écoute.
Mode filaire : utilisez le câble jack même si la batterie est vide.
